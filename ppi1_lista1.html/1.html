Internet x Web

A Internet é a rede mundial de computadores que conecta dispositivos no mundo todo.
Já a Web é um serviço que funciona dentro da internet, usado para acessar sites.
Ou seja, a internet é a estrutura e a web é o que usamos para navegar.
Sem internet não existe web, mas a internet tem outros serviços além da web.

História da Internet e da Web

A história começou na década de 1960 com projetos militares.

1969 - Criação da ARPANET
1983 - Uso do protocolo TCP/IP
1991 - Criação da Web por Tim Berners-Lee

A Web surgiu para facilitar o acesso às informações usando navegadores.
