O que é HTTP

HTTP é um protocolo usado na web para comunicação entre o navegador e o servidor. Ele funciona com requisição (pedido) e resposta.

Como funciona

Quando você entra em um site, o navegador envia uma requisição e o servidor responde com a página.

Exemplo de Request

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Mozilla
Accept: text/html

GET = tipo de requisição
Host = endereço do site
User-Agent = navegador
Accept = tipo de resposta

Exemplo de Response

HTTP/1.1 200 OK
Content-Type: text/html

<html>
<body>
Olá mundo
</body>
</html>

200 OK = deu certo
Content-Type = tipo do conteúdo
